Um **resumo simples e direto** dos comandos que você usou para subir o código para o GitHub:

---

## ✅ **Resumo dos comandos utilizados**

### **1. Inicializar o repositório Git**

```bash
git init
```

Cria um repositório Git vazio na pasta do projeto.

---

### **2. Adicionar o repositório remoto**

```bash
git remote add origin https://github.com/lucinhalsd/App_Hobby.git
```

Conecta o projeto local ao repositório no GitHub.

---

### **3. Criar/renomear a branch principal para “main”**

```bash
git branch -M main
```

Define a branch atual como `main`.

---

### **4. Adicionar todos os arquivos ao controle de versão**

```bash
git add .
```

Coloca todos os arquivos modificados/criados na “staging area”.

---

### **5. Criar o commit**

```bash
git commit -m "first commit"
```

Salva as alterações localmente com a mensagem “first commit”.

---

### **6. Enviar o código para o GitHub**

```bash
git push -u origin main
```

Envia os arquivos commitados para a branch `main` no GitHub e define o rastreamento automático dessa branch.
